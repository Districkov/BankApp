## 🔌 API Интеграция

### Эндпоинты

**Авторизация:**
```
GET  /api/auth/simple/yandex/url
POST /api/auth/simple/yandex/callback
GET  /api/auth/whoami
POST /api/auth/logout
```

**Счета:**
```
GET /api/accounts/accounts
GET /api/accounts/accounts/:id
GET /api/accounts/accounts/:id/balance
```

**Переводы:**
```
POST /api/transfers/transfers/phone
POST /api/transfers/transfers/card
POST /api/transfers/transfers/internal
```

**Транзакции:**
```
GET /api/accounts/transactions
GET /api/accounts/transactions/:id
```

### Антифрод проверки (на бэкенде)

**Лимиты:**
- Одиночный перевод: до 600,000 ₽
- Дневной лимит: до 1,000,000 ₽
- Месячный лимит: до 5,000,000 ₽

**Проверки:**
- Подозрительная активность
- Множественные переводы
- Круглые суммы
- Новые получатели
