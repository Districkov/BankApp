## 🔌 API Интеграция

### Эндпоинты
```
Auth:     /api/auth/v1/*
Accounts: /api/accounts/v1/*
Transfers: /api/transfers/v1/*
```

### Антифрод коды ошибок
- **400**: Некорректные данные
- **403**: Превышен лимит / Подозрительная активность
- **429**: Слишком много запросов
